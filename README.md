# 💸 Conto

**De expense splitter voor je maten.** Geen accounts, geen gedoe — gewoon uitgaven droppen en splitten.

👉 **[Open Conto](https://cedricbaertdelaware.github.io/Conto/)**

---

## Wat is Conto?

Conto is een web app om gedeelde uitgaven bij te houden tijdens trips. Gebouwd voor een Comomeer trip met 8 maten, maar werkt voor elke groepsreis.

Denk Splitwise of Tricount, maar dan sneller, grappiger, en zonder account.

## Features

- **Real-time sync** — Iedereen ziet dezelfde data via Firebase. Iemand dropt een uitgave? Verschijnt meteen bij de rest.
- **Snel uitgaven toevoegen** — Bedrag, beschrijving, categorie, klaar. Geoptimaliseerd voor één-hand gsm-gebruik.
- **Wie is blut?** — Overzicht van wie in plus of min staat, met jouw persoonlijke saldo bovenaan.
- **Time to pay** — Minimale transacties berekend om alles te vereffenen. IBAN kopiëren met één tik.
- **Betalingen markeren** — Tik "Betaal" als je overgeschreven hebt. Iedereen ziet het live.
- **Categorieën** — Fret, Drankie, Vroem, Slapen, Activiteit, Random.
- **Dark mode** — Automatisch op basis van je systeeminstellingen.
- **Straattaal UI** — Omdat geld splitten niet saai hoeft te zijn.
- **Easter egg** — Tik 5x snel op je saldo kaart. 🎉

## Hoe het werkt

1. Open de link op je gsm
2. Kies wie je bent uit de squad
3. Drop uitgaven, check balansen, reken af
4. Deel de link met je maten in de groepschat

Geen downloads, geen accounts, geen installatie. Gewoon een link.

## Tech stack

- Vanilla HTML/CSS/JS (single file, ~25KB)
- Firebase Realtime Database (gratis tier)
- Gehost op GitHub Pages
